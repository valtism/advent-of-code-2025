# Advent of Code Runner

A runner for Advent of Code challenges. To get started, add your session token `AOC_SESSION_KEY` to the `.env` file. You can find this in your browser's cookies after logging into the Advent of Code website.

Then, simply run:

```bash
bun run start
```

You will need to have Bun installed on your system. If you don't have it installed, you can download it from [here](https://bun.sh/).

---

This utility will automatically download the input data for each day and run the corresponding solution in a watch mode.
